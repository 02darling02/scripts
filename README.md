SSPANEL 面板签到程序

使用 GitHub Actions 来完成操作.

本想用 JAVA 的，但是这么一个小小的东西，决定还是用 JavaScripts，目前仅支持单账号单网站！

## *Env*

|    Name     |                 Value                 |
| :---------: | :-----------------------------------: |
| SSPANEL_API | 要签到的网站，e.g：https://www.xxx.me |
|    EMAIL    |                 邮箱                  |
|   PASSWD    |                 密码                  |
